# How To Run
1. Download or Clone Project
2. Navigate To The Root of The Project
3. Open Terminal
    For Windows, Type ``` py main.py ``` & Press Enter
    For Windows & Mac, Type ``` python3 main.py ``` & Press Enter

Make Sure Python3 Is Installed On Your System
